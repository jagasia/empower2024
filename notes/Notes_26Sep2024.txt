Create an EC2 linux instance:
	EC2 Dashboard:
		Instances -> Launch an Instance
	i) enter a name for the instance
	ii) Choose OS as Amazon Linux (default) (free tier eligible)
	iii) Choose Key pair or create a new key pair. (the .pem file will be downloaded to downloads folder)
	iv) Network settings (check the "Auto-assign public IP)
		edit and enable the auto assign public ip
	v) Select existing security group (if already exists)

Finally, click the "Launch Instance" button.
Check the dashboard for instances and refresh to see the status changing from "Pending" to "Running".

When it is running, click the instance id (hyperlink).
Click connect button.

To install docker in AWS EC2 linux instance:

sudo yum update

sudo yum search docker

sudo yum install docker

sudo usermod -a -G docker ec2-user
id ec2-user

sudo systemctl enable docker.service

sudo systemctl start docker.service

sudo systemctl status docker.service
